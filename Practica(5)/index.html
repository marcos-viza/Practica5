¿Que es HTML?

el lenguaje fundamental que se utiliza para la creación de páginas en la World Wide Web. Su función principal es servir como la "columna vertebral" de un sitio, permitiendo organizar el contenido a través de una estructura de etiquetas y contenedores, como la etiqueta, que agrupa diferentes objetos dentro de la página.

¿Que es CSS?

es un lenguaje de diseño gráfico que se utiliza para definir y crear la presentación de un documento estructurado (como el HTML)

¿Que es JS?

JavaScript es un lenguaje de programación que se utiliza para construir sitios web y hacerlos mucho más interactivos
